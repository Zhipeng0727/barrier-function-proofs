1.2 Self-Evolve V2 Agent

NEW self_evolve_v2.py — 自我迭代进化代理（~320 行）

六步闭环：

ROUND N:
  EXPLORE   → attempt_proof on frontier (按 recipe-gap 优先级排序)
  ANALYZE   → 分类失败 (hallucinated_lemma / arithmetic_gap / type_class)
  MINE      → extract_obstruction → mine_recipe (GLM→GPT 自动接力)
  VERIFY    → lean_compile_check oracle (zero false positive)
  RE-INJECT → 用新 recipe 重试失败系统
  CONVERGE  → 连续 K_DRY=3 轮无新 recipe 则停止

关键特性：

  • 双 Provider 接力：GLM-4.7 primary → GPT-5.5 fallback（via aigocode relay）
  • 前沿评分：score = recipe_overlap + 0.3*poly + 0.2*untested + 0.1*transcendental
  • 崩溃恢复：JSON checkpoint + --resume flag
  • CLI：--max-rounds / --k-dry / --budget / --untested-only / --no-escalate
